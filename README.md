jQuery ajaxHooks
================

What is it?
-----------

jQuery ajaxHooks is a collection of `jQuery.ajax`, `jQuery.Callbacks` and `jQuery.Deferred` extensions and utilities.

See [here](https://github.com/jaubourg/ajaxHooks/blob/master/doc/BUILD.md).
